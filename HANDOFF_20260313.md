### 6. Critical Constraints Confirmed
- **tau_osc = 10ms** - MUST be ≤10ms, higher suppresses oscillation
- **A_osc ≈ tau_m × 1.5** - Scales with membrane time constant
- **method='euler'** - Brian2 bug #626: exponential_euler fails with sin/cos
- **State variables need explicit assignment** - NOT automatic from namespace

### Single Oscillatory System Test
```bash
source .venv/bin/activate && python3 -c "
from core.neurons import OscillatoryNeuron
from brian2 import ms, mV, Hz, start_scope, run, SpikeMonitor
import numpy as np

start_scope()
ng = OscillatoryNeuron.create_group(
    N=50, frequency=10*Hz, name='test',
    params={'tau_m': 20*ms, 'tau_osc': 10*ms, 'A_osc': 30*mV}

print(f'Spikes: {spikes.num_spikes}, Rate: {spikes.num_spikes/(50*0.5):.1f} Hz/neuron')
"
```
